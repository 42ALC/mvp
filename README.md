# mvp
minimum viable product 를 통해 진행 여부를 결정해보자

## mvp 기획단

kilee, hekang, ekim, hyeonkim, mijeong

## 핵심 목표

동료학습 활성화에 기여할 수 있는 체계 구축을 위한 시범 운영

## 계획

### mvp구현 목표

1. 시범 서비스를 위한 채널 개설 (카카오톡 오픈 채팅방)

2. 채팅 방내에서 에티켓 규칙
3. 표준화된 대면/비대면 동료학습 세션 몇가지 만들기
4. 시범서비스 운영진 모집

#### 1. 시범 서비스를 위한 채널 개설

플랫폼 : 카카오톡 오픈채팅방

구조

```
.
|- 라운지방
	|_ 0층(0 circle)
	|_ 1층(1 circle)
	|_ 2층(2 circle)
	|_ 3층(3 circle)
	|_ 4층(4 circle)
	|_ 5층(5 circle)
	|_ 6층(6 circle)
```
